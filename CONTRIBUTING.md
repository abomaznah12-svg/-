# إرشادات المساهمة

شكراً لاهتمامك بالمساهمة في هذا المشروع!

## كيفية المساهمة

### الإبلاغ عن المشاكل

إذا وجدت مشكلة، يرجى:
1. التحقق من أن المشكلة لم يتم الإبلاغ عنها مسبقاً
2. إنشاء issue جديد مع وصف واضح للمشكلة
3. تضمين خطوات إعادة إنتاج المشكلة إن أمكن

### إرسال التغييرات

1. قم بعمل Fork للمستودع
2. أنشئ فرع جديد للميزة أو الإصلاح:
   ```bash
   git checkout -b feature/اسم-الميزة
   ```
3. قم بإجراء التغييرات المطلوبة
4. تأكد من أن التغييرات تعمل بشكل صحيح
5. قم بعمل commit للتغييرات مع رسالة واضحة:
   ```bash
   git commit -m "وصف موجز للتغييرات"
   ```
6. ادفع التغييرات إلى فرعك:
   ```bash
   git push origin feature/اسم-الميزة
   ```
7. أنشئ Pull Request

## معايير الكود

- اتبع أسلوب الكود الموجود في المشروع
- اكتب تعليقات توضيحية عند الحاجة
- تأكد من أن الكود يعمل بدون أخطاء
